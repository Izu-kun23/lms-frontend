# Shared Components

This directory contains components shared across multiple user roles.

## Current Components

- `app-shell.tsx` - Main application shell with sidebar and layout
- `login-form.tsx` - Public login form (used by students/instructors)
- `signup-form.tsx` - Public signup form
- `providers.tsx` - QueryClient provider wrapper
- `theme-provider.tsx` - Theme provider for light/dark mode
- `user-menu.tsx` - User profile menu component

## Usage

These components are used across the application by all user roles.
